Welcome to AutoPal Financial!

    Hello and welcome to Autopal Financial! We created this website in hopes of spreading financial literacy by offering services such as our auto loan calculator, budget assistant, and financial API resources. This project was created by Alberto Del Real, Thomas West and Weston Thai for Arizona State University's Informational Technologies Capstone Project.

Services

  • Auto Loan Calculator
  • Real Time API Resources
  • Budget Assistant
  • Login Capabilities
    This web application features our understanding of Flask, Python, HTML, CSS and MySQL. We learned how to integrate APIs and back-end services through Flask, MVC architecture for user experience, and version control efficiency. We hope this application provides you assistance for your financial needs!
